# Library-management-system
Task management system
Library Management System:-
This is a task management system that can be used to solve real-world problems such as managing books in a library or tracking tasks efficiently.

Project Description:- 
This project is a simple Library Management System developed using Python. It allows users to add, view, issue, and delete books. The system stores data permanently using a JSON file.

This project is beginner-friendly and demonstrates:
Functions:
* Loops
* Lists and dictionaries
* File handling using JSON
* User input validation
  
Features:
* Add new books to the library
* View all books in a table format
* Issue books with user-entered issue date
* Prevent deletion of issued books
* Save and load data using books.json
* Menu-driven console application
  
Technologies Used:
* Python 3
* JSON file handling
  
Files Explanation:
* library-management-system.py : Main program file containing all logic
* books.json : Stores library data (created automatically)

How to Run the Project:-
1. Make sure Python is installed on your system.
2. Clone or download this repository.
3. Open terminal / command prompt.
4. Run the program using: python library-management-system.py
   
Menu Options:
1. Add Book
2. View Books
3. Issue Book (Enter Issue Date)
4. Delete Book
5. Exit

Rules Implemented:-
1. Issued books cannot be deleted
2. Issue date must be entered by the user
3. Only numeric input is allowed for menu options
